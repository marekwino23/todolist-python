# todolist-python
Python CLI TODO App – Prosta aplikacja w terminalu do zarządzania codziennymi zadaniami. Funkcjonalności:  Dodawanie, usuwanie i wyświetlanie zadań  Oznaczanie zadań jako ukończone lub w trakcie  Czytelny interfejs w terminalu  Dane są przechowywane w pamięci (możliwość rozszerzenia o zapis do pliku JSON)
